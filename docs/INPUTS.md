# Input Mappings

## Player Actions

| Action | Key | Button | Description |
|--------|-----|--------|-------------|
| move_up | W | Left Stick Up | Move player up |
| move_down | S | Left Stick Down | Move player down |
| move_left | A | Left Stick Left | Move player left |
| move_right | D | Left Stick Right | Move player right |
| attack | Left Click | Right Trigger | Primary attack |
| special | Right Click | Left Trigger | Special ability |
| interact | E | South Button | Interact with objects |
| pause | Escape | Start | Pause game |

## UI Navigation

| Action | Key | Button | Description |
|--------|-----|--------|-------------|
| ui_accept | Enter | South Button | Confirm selection |
| ui_cancel | Escape | East Button | Cancel/Back |
| ui_up | Up Arrow | D-Pad Up | Navigate up |
| ui_down | Down Arrow | D-Pad Down | Navigate down |
| ui_left | Left Arrow | D-Pad Left | Navigate left |
| ui_right | Right Arrow | D-Pad Right | Navigate right |

## Configuration

Input mappings are configured in Project Settings > Input Map.
See `resources/configs/input_config.tres` for runtime configuration.
