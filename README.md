# RadGo

A Godot 4 game project.

## Project Structure

This project follows a modular architecture with clear separation of concerns:

- **scenes/** - Game scenes organized by type
- **scripts/** - GDScript files organized by functionality
- **resources/** - Game data and configurations
- **assets/** - Visual and audio assets
- **docs/** - Project documentation

## Getting Started

1. Open `project.godot` in Godot 4.x
2. Configure your input settings in Project Settings
3. Run the main scene from `scenes/main/`

## Development

See `docs/` for detailed documentation:
- `docs/ARCHITECTURE.md` - System architecture
- `docs/GAME_DESIGN.md` - Gameplay mechanics
- `docs/STYLE_GUIDE.md` - Coding conventions
- `docs/TASKS.md` - Current development tasks

## Building

Export presets are configured in `export_presets.cfg`. Builds are output to `builds/` directory.
